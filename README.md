# Green Logistics & Carbon Footprint Optimizer

## Overview
This project focuses on optimizing supply chain operations by analyzing the carbon footprint of warehouse logistics. By leveraging data analytics, the project identifies high-emission routes and simulates the impact of transitioning from traditional fuel (Diesel) to sustainable alternatives (Electric/Hybrid).

## Technical Stack
*   **Python:** Pandas, Matplotlib, Io.
*   **SQL:** Integrated via `pandasql` for data manipulation.
*   **Methodology:** Data aggregation, KPI calculation, and comparative analysis.

## Key Features
*   **Data-Driven Decision Making:** Calculated the `Total_Carbon_Footprint` as a key KPI for logistics efficiency.
*   **SQL Integration:** Performed advanced data filtering and aggregation using standard SQL syntax within a Python pipeline.
*   **Sustainability Insights:** Visualized the carbon emission gap between Diesel and Electric transportation modes, providing a clear roadmap for emission reduction.

## Analysis Summary
The project demonstrates that:
*   **Diesel** transport remains the primary source of carbon emissions in the current warehouse network.
*   Transitioning to **Electric** transportation yields a significant reduction in the average carbon footprint (from ~103kg to ~6.75kg per route).

## How to Use
1. Clone the repository.
2. Ensure you have the required libraries installed: `pip install pandas pandasql matplotlib`.
3. Run the script to generate the sustainability analysis report and visualizations.
